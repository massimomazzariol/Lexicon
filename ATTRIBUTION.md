# Attribution Guide

If you reuse Lexicon Platform code, data, or derived materials, please keep the
license notices intact and provide visible attribution.

## Preferred Attribution

For lexical content, packs, and derived distributions:

> Lexicon Platform lexical data by Massimo Mazzariol and contributors,
> licensed under CC BY 4.0.

For software integrations or code reuse:

> Uses Lexicon Platform by Massimo Mazzariol and contributors.

## Include These Links

- Repository: https://github.com/massimomazzariol/Lexicon
- Code license: `LICENSE`
- Content license: `docs/CONTENT_LICENSE.md`

## If You Modified The Material

Please indicate that your version includes modifications, additions, or
editorial changes. A short note is enough, for example:

> Adapted from Lexicon Platform. Modified entries and added local extensions.
